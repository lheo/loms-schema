learningOutcomeType element
===========================

The learning outcome type (i.e. knowledge, skill, ...).

.. code-block:: xml

  <learningOutcomeType uri="...">
  </learningOutcomeType>

Restricts: :ref:`uribasedcode-type`

Attributes
-----------
