.. _webresource-element:

webresource element
===================

A contact form used for contacting the agent.

.. code-block:: xml

  <webresource uri="...">
  </webresource>

Attributes
-----------

``uri``: ``required`` `xs:anyURI <https://www.w3.org/TR/xmlschema11-2/#anyURI>`_
	The URL to access the interactive web resource.
