IdReference
===========

A link or reference to another entity record in the credential document.


.. list-table::
    :widths: 25 25 50
    :header-rows: 1

    * - Attribute
      - Type
      - Description
    * - ``idref``
      - ``required`` `xs:anyURI <https://www.w3.org/TR/xmlschema11-2/#anyURI>`_
      - The id of the referenced entity (record).
